Clases

En programación orientada a objetos, las clases son como plantillas o moldes para crear objetos. Los objetos son instancias de una clase y pueden tener propiedades (llamadas atributos) y comportamientos (llamados métodos).

Verás que en los ejemplos que daremos a continuación ya incluimos elementos como objetos y constructores. No te preocupes, los cubriremos más a detalle en las siguientes lecciones.


Python

# Definición de una clase en Python
class Perro:
    # Constructor con atributos
    def __init__(self, nombre, raza):
        self.nombre = nombre
        self.raza = raza

    # Método para imprimir información del perro
    def imprimir_informacion(self):
        print(f"Nombre: {self.nombre}, Raza: {self.raza}")

# Crear un objeto (instancia) de la clase Perro
mi_perro = Perro(nombre="Fido", raza="Labrador")

# Acceder a los atributos y llamar a un método
print(mi_perro.nombre)
mi_perro.imprimir_informacion()

Java

// Definición de una clase en Java
public class Perro {
    // Atributos
    String nombre;
    String raza;

    // Constructor con parámetros
    public Perro(String nombre, String raza) {
        this.nombre = nombre;
        this.raza = raza;
    }

    // Método para imprimir información del perro
    public void imprimirInformacion() {
        System.out.println("Nombre: " + nombre + ", Raza: " + raza);
    }

    // Método principal (main) para probar la clase
    public static void main(String[] args) {
        // Crear un objeto (instancia) de la clase Perro
        Perro miPerro = new Perro("Fido", "Labrador");

        // Acceder a los atributos y llamar a un método
        System.out.println(miPerro.nombre);
        miPerro.imprimirInformacion();
    }
}
