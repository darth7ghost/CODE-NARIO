Operadores

Los operadores en Python y Java son símbolos especiales que se utilizan para realizar operaciones en variables y valores.


Python

Operadores Aritméticos: Estos operadores se utilizan para realizar operaciones matemáticas básicas.

  • + : Suma.
  • - : Resta.
  • * : Multiplicación.
  • / : División.
  • % : Módulo (obtiene el resto de una división).
  • ** : Exponente (elevar a una potencia).

a = 5
b = 2

suma = a + b  # suma = 7
resta = a - b  # resta = 3
multiplicacion = a * b  # multiplicacion = 10
division = a / b  # division = 2.5
modulo = a % b  # modulo = 1
exponente = a ** b  # exponente = 25

Operadores de Comparación: Estos operadores se utilizan para comparar valores y devuelven un resultado booleano (True o False).

  • == : Igual a.
  • != : Diferente de.
  • < : Menor que.
  • > : Mayor que.
  • <= : Menor o igual que.
  • >= : Mayor o igual que.

x = 5
y = 10

igual = x == y  # igual = False
diferente = x != y  # diferente = True
menor = x < y  # menor = True
mayor = x > y  # mayor = False

Java

Java utiliza operadores similares a Python, aunque la sintaxis puede diferir un poco. Aquí están los operadores más comunes:

Operadores aritméticos:

  • + : Suma.
  • - : Resta.
  • * : Multiplicación.
  • / : División.
  • % : Módulo (obtiene el resto de una división).

int a = 5;
int b = 2;

int suma = a + b;  // suma = 7
int resta = a - b;  // resta = 3
int multiplicacion = a * b;  // multiplicacion = 10
int division = a / b;  // division = 2
int modulo = a % b;  // modulo = 1

Operadores de comparación:

  • == : Igual a.
  • != : Diferente de.
  • < : Menor que.
  • > : Mayor que.
  • <= : Menor o igual que.
  • >= : Mayor o igual que.

int x = 5;
int y = 10;

boolean igual = x == y;  // igual = false
boolean diferente = x != y;  // diferente = true
boolean menor = x < y;  // menor = true
boolean mayor = x > y;  // mayor = false
