Variables y tipos de datos

Las variables y tipos de datos son conceptos fundamentales en la programación, tanto en Python como en Java.


Python

En Python, una variable es un contenedor que puedes usar para almacenar datos. Puedes crear una variable simplemente dando un nombre a la misma y asignándole un valor. Por ejemplo: 

mi_variable = 42

Python es un lenguaje de programación con tipado dinámico, lo que significa que no necesitas declarar explícitamente el tipo de dato de una variable. Algunos tipos de datos comunes en Python incluyen:

  • int: Números enteros (por ejemplo, 5).
  • float: Números de punto flotante (por ejemplo, 3.14).
  • str: Cadenas de texto (por ejemplo, "Hola, mundo").
  • bool: Valores booleanos (True o False).

mi_numero = 5
mi_texto = "Hola, mundo"

Java

En Java, las variables también son contenedores para almacenar datos. Debes declarar el tipo de variable y luego asignarle un valor. Por ejemplo:

int miVariable = 42;

Java es un lenguaje de tipado estático, lo que significa que debes declarar el tipo de dato de una variable antes de usarla. Algunos tipos de datos comunes en Java incluyen:

  • int: Números enteros (por ejemplo, 5).
  • double: Números de punto flotante de doble precisión (por ejemplo, 3.14).
  • String: Cadenas de texto (por ejemplo, "Hola, mundo").
  • boolean: Valores booleanos (true o false).

    • int miNumero = 5;
String miTexto = "Hola, mundo";
